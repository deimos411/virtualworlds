# Virtual Worlds  
Virtual worlds I made in 1998 in VRML with Internet Space Builder 2.1 and Cosmo World. 
Recently restaured using Blender 2.9 and BabylonJS.  

New!
* Better XBox gamepad support.
* Mobile navigation improvement.
